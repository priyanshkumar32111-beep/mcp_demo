# mcp_demo

This repository demonstrates MCP (Model Context Protocol) integration with GitHub Copilot and Jira.

## Features
- Business Requirements Document (BRD) for an Online Bookstore
- Automated extraction of requirements and user stories
- Jira issue creation from user stories

## Structure
- `brd.md`: Business Requirements Document
- `README.md`: Project overview and usage

## Usage
1. Clone the repository:
   ```sh
   git clone https://github.com/priyanshkumar32111-beep/mcp_demo.git
   ```
2. Review the BRD in `brd.md`.
3. Use Copilot agent workflows to extract requirements and sync with Jira.

## Integration
- MCP servers are configured in `.vscode/mcp.json` (hosted, no local server needed).
- Jira project: `mcp_demo` (key: MD)
